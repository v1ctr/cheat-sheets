# PostgreSQL Cheat Sheet

## Command Line (psql)

### Listing all databases
    \l
### Creating Database
    CREATE DATABASE <database-name>;
### Connecting to a database
    \c <database-name>
### Listing all tables in current database
    \d
### Exiting the Command Line
    \q
### Executing SQL
Just enter the SQL Command e.g.:
    
    SELECT * FROM users;
